# CLAUDE.md — Инструкции для AI-агента

## Дополнительные инструкции (сабмодуль `claude/`)

> 📚 **Обязательно зайти в директорию сабмодуля [`claude/`](claude/) и прочитать все `.md`-файлы оттуда** (например, `claude/CLAUDE.base.md`, `claude/CLAUDE.python.md`).
> Эти файлы содержат общие правила и инструкции, которые также должны соблюдаться в этом проекте.
> Сабмодуль подключён из репозитория https://github.com/vadim-kosarev/claude.git

## Правила работы с git

> ⛔ **Не коммитить изменения и не пушать в git без явного указания пользователя.**
> Вносить правки в файлы можно, но `git commit` / `git push` выполнять только после явной команды.

## Деплой изменений на серверы

> ⛔ **Не заливать файлы на серверы напрямую через pscp или scp.**
> Все изменения конфигурации деплоятся через git:
> 1. Внести правки локально
> 2. Закоммитить (по указанию пользователя)
> 3. Пользователь сам пушит в репозиторий
> 4. На сервере выполнить `git pull` и при необходимости `docker compose up -d`

## Контекст проекта

Этот репозиторий предназначен для управления конфигурациями VPS хостингов.  
Каждая директория верхнего уровня соответствует отдельному серверу или домену.

> 📖 **Описание проекта — в файле [`README.md`](README.md).** Перед началом работы обязательно прочитать его.
