# vkosarev.link — описание хостинга

## SSH-подключение

- Пользователь для SSH: `ubuntu`
- Сервер поддерживает вход по ключу (рекомендуется)
- Для подключения используйте PuTTY/plink или стандартный ssh-клиент:

```powershell
plink -load "vkosarev.link" -i "Z:\MY\vk-amazon-2023-private.ppk" -batch "команда"
```
или
```bash
ssh -i ~/.ssh/your-key ubuntu@vkosarev.link
```

## Особенности
- Сервер поддерживает команды управления через SSH.
- Все основные сервисы управляются через Docker Compose.
