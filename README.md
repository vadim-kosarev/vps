# VPS Hosting Configurations

Репозиторий содержит конфигурации для разворачивания сервисов на различных VPS хостингах.

## Структура проекта

Каждая папка соответствует отдельному хостингу (домену или серверу):

```
vps/
├── vkosarev.name/        # Конфигурация для хостинга vkosarev.name
│   └── docker-compose.yml
└── ...                   # Другие хостинги
```

## Принцип организации

- Каждая директория верхнего уровня — отдельный VPS или домен.
- Внутри каждой директории находятся файлы конфигурации для развёртывания сервисов (преимущественно `docker-compose.yml`).
- Конфиденциальные данные (пароли, токены, `.env`-файлы) **не хранятся** в репозитории.

## Развёртывание

Переходим в нужную директорию и запускаем:

```bash
cd <имя-хостинга>
docker compose up -d
```

## Требования

- [Docker](https://docs.docker.com/get-docker/) >= 20.x
- [Docker Compose](https://docs.docker.com/compose/) >= 2.x (плагин `docker compose`)
