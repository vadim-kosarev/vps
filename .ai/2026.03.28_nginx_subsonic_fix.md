# 2026.03.28_nginx_subsonic_fix.md

## Изменения для корректной работы Subsonic на vkosarev.name

**Дата:** 2026-03-28

### Кратко
- Исправлен nginx-конфиг для Subsonic (порт 4041):
  - Добавлен `server_name vkosarev.name;` в server-блок для 4041.
  - Это устраняет ошибочный редирект на https://vkosarev.name:1443/login.view? при обращении к http://vkosarev.name:4041.
- Причина: без server_name nginx не мог однозначно сопоставить запросы на порт 4041, и по умолчанию применялся редирект с порта 80 (default_server).
- Теперь Subsonic доступен по http://vkosarev.name:4041 без редиректов на https.

### Хронология действий
1. Проанализирован nginx-конфиг и логика обработки портов.
2. Установлена причина редиректа: отсутствие server_name в server-блоке для 4041.
3. Внесено исправление: добавлен server_name vkosarev.name.
4. Подготовлен и выполнен коммит изменений.
5. Дано указание перезапустить nginx для применения изменений.
