# 2026.03.28_windows_exporter_setup_report.md

## Задача

Автоматизировать установку и запуск Prometheus windows_exporter на Windows-хосте с помощью PowerShell-скрипта `setup.ps1`, а также исключить бинарник экспортера из репозитория через `.gitignore`.

---

### Хронология и шаги

1. **Формат отчёта**
   - Все строки отчёта в скрипте теперь имеют формат: ` - Имя шага : результат`.

2. **Корректное экранирование аргументов**
   - Исправлено формирование строки `$binArgs` для корректной передачи пути и аргументов в `New-Service`.

3. **Информативность отчёта**
   - В отчёт об установке/перезапуске windows_exporter добавлен URL для просмотра метрик: `http://localhost:9182/metrics`.

4. **Исключение бинарника из репозитория**
   - В `.gitignore` добавлен путь `bin/windows_exporter.exe`.

---

## Итоговое состояние

- Скрипт `setup.ps1`:
  - Создаёт пользователя `local-admin` (если не существует), добавляет в администраторы.
  - Включает WinRM, настраивает UAC bypass.
  - Скачивает windows_exporter.exe в папку `bin` (если отсутствует).
  - Устанавливает windows_exporter как сервис с конфигом.
  - В отчёте указывает URL для проверки метрик.
  - Все этапы сопровождаются отчётом в формате ` - Имя шага : результат`.

- `.gitignore`:
  - Исключает `bin/windows_exporter.exe` из индексации git.
