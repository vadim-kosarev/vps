## nginx: основные порты и логика

- 80      — HTTP, редирект на 1443 (HTTPS)
- 1443    — HTTPS, статика (index.html), автоиндекс /a/
- 7601    — редирект на https://vkosarev.name:7601 (Immich)

## frps
- 7401/tcp+udp — основной порт для клиентов FRP
- 7599         — dashboard FRP

## 3x-ui
- 443, 8888, 22000 — VLESS/Reality/HTTP-proxy (точные назначения см. в панели)
- 33562 — web-панель управления (https)

## openvpn
- 29817 — внешний порт, проброшен в контейнер (1194/tcp)

## portainer
- 8000  — API (Edge Agent)
- 9443  — UI (https)
