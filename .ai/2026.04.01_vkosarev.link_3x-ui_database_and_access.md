# 2026-04-01 — Информация о доступе к 3x-ui на vkosarev.link

## Где лежит БД 3x-ui

**Путь на хосте:**
```
/root/vps/vkosarev.link/3x-ui/db/x-ui.db
```

**Тип:** SQLite 3.x база данных

## Установка SQLite клиента

Была установлена утилита `sqlite3` на сервер vkosarev.link:
```bash
sudo apt-get update && sudo apt-get install -y sqlite3
```

## Таблицы в БД

| Таблица | Описание |
|---|---|
| `users` | Пользователи панели |
| `settings` | Конфигурация панели (порты, пути, сертификаты) |
| `inbounds` | Входящие соединения (VLESS, Reality и т.д.) |
| `client_traffics` | Статистика трафика клиентов |
| `outbound_traffics` | Статистика исходящего трафика |
| `inbound_client_ips` | IP адреса клиентов |
| `history_of_seeders` | История сидеров |

## Доступ к панели 3x-ui

### URL панели
```
https://vkosarev.link:33562/vkosarev.link.amazon/panel
```

или

```
https://vkosarev.link:33562/vkosarev.link.amazon/panel/settings
```

| Параметр | Значение |

## Проблема входа

**Если пароль забыт,** можно сбросить через CLI 3x-ui:

```bash
# Установить новый пароль (пример: NewPassword123)
sudo docker compose -f /root/vps/vkosarev.link/docker-compose.yml exec -T 3x-ui \
  /app/x-ui setting -username vkosarev -password "NewPassword123"

# Проверить текущие настройки
sudo docker compose -f /root/vps/vkosarev.link/docker-compose.yml exec -T 3x-ui \
  /app/x-ui setting -show
```

## Состояние портов

Все необходимые порты слушают:

| Порт | Процесс | Назначение |
|---|---|---|
| **33562/tcp** | x-ui | Веб-панель управления (HTTPS) |
| **443/tcp** | xray-linux | VLESS+Reality |
| **8888/tcp** | xray-linux | VLESS+Reality (альтернативный) |
| **22000/tcp** | xray-linux | HTTP proxy |

## Команды для работы

### Просмотр пользователей
```bash
cat > /tmp/query.sql << 'EOF'
SELECT id, username FROM users;
EOF
sudo sqlite3 /root/vps/vkosarev.link/3x-ui/db/x-ui.db < /tmp/query.sql
```

```bash
sudo docker compose -f /root/vps/vkosarev.link/docker-compose.yml exec -T 3x-ui \
```
