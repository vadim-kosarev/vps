# 2026-04-01 — Исправление ошибки загрузки сертификатов 3x-ui на vkosarev.link

## Проблема
Контейнер `3x-ui` на `vkosarev.link` запускался, но не загружал HTTPS-сертификаты, работая только в HTTP режиме на порту 33562.

**Ошибка в логах:**
```
ERROR - Error loading certificates: open /root/cert/fullchain.pem: no such file or directory
```

## Корневая причина
Файлы сертификатов имели неправильные имена:
- 3x-ui ожидал: `/root/cert/fullchain.pem`
- На диске находился: `/root/cert/vkosarev.link.fullchain.pem`

Второй файл ключа был правильно назван:
- ✅ `/root/cert/vkosarev.link_privatekey.key` — уже соответствует ожиданиям

## Решение
Создана символическая ссылка:
```bash
sudo ln -sf /root/cert/vkosarev.link.fullchain.pem /root/cert/fullchain.pem
```
