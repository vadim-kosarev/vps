# Чеклист безопасности для хоста vkosarev.name (VPS)

_Дата генерации: 2026-03-28_

## 1. Пароли и секреты
- [ ] Все пароли и токены (Grafana, 3x-ui, frps, Portainer, Prometheus и др.) уникальны и сложны, не используются значения по умолчанию (changeme, admin, 1234 и т.п.)
- [ ] Секреты (токены, пароли) не хранятся в открытых файлах репозитория (frps.toml, .env, docker-compose.yml)
- [ ] Для секретов используются Docker secrets или переменные окружения вне репозитория
- [ ] Пароль в nginx/.htpasswd сложный, не dictionary-based

## 2. Открытые порты и сервисы
- [ ] Минимизировано количество проброшенных наружу портов (особенно для административных панелей)
- [ ] Административные интерфейсы (3x-ui, Grafana, Prometheus, Portainer, frps dashboard) доступны только с доверенных IP (через firewall или nginx allow/deny)
- [ ] Portainer открыт только для localhost/VPN, API недоступен извне

## 3. Docker best practices
- [ ] Для всех production-контейнеров используются фиксированные версии образов (не latest)
- [ ] Не используется network_mode: host без крайней необходимости
- [ ] Доступ к docker.sock ограничен только необходимыми сервисами

## 4. SSH и root-доступ
- [ ] Отключён root-доступ по SSH или ограничен по IP
- [ ] Разрешена только ключевая аутентификация, парольный вход отключён
- [ ] Включён fail2ban для защиты от перебора

## 5. frps
- [ ] Секреты (auth.token, webServer.password) уникальны и сложны
- [ ] Dashboard frps (порт 7599) доступен только с доверенных IP

## 6. nginx
- [ ] Включены заголовки безопасности: HSTS, X-Frame-Options, X-Content-Type-Options, Content-Security-Policy
- [ ] Все чувствительные location защищены авторизацией
- [ ] Отключены ненужные HTTP-методы

## 7. Мониторинг и аудит
- [ ] Включён аудит логинов и действий (auditd, fail2ban, мониторинг логов)
- [ ] Регулярно обновляются образы и зависимости

## 8. Обновления и CVE
- [ ] Регулярно проверяются CVE для используемых образов и своевременно обновляются
