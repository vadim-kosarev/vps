Панель 3x-ui не открывалась, хотя контейнер был запущен. Логи показывали первоначально HTTP вместо HTTPS.

## Решение

### Проблема была в редиректе
URL `/vkosarev.link.amazon/panel` (без слэша) перенаправляется на `/vkosarev.link.amazon/` 
(HTTP 307 Temporary Redirect)

### Правильный URL панели
```
https://vkosarev.link:33562/vkosarev.link.amazon/panel/
```

**С слэшом на конце!**
